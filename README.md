# Installation
* Using IntelliJ IDEA
* Add `VM options`:
  * `Run` -> `Edit Configurations...`
    -> `Modify options` -> `Add VM options`:
    > --module-path /path/to/javafx-sdk-17/lib
    > --add-modules
    > javafx.controls
    > --add-modules
    > javafx.fxml
* `Run` -> `Run BombermanGame`
